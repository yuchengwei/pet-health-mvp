# 宠物健康管理+社区 - UI设计说明文档

**版本：** v1.0  
**设计对象：** 微信小程序 MVP  
**设计日期：** 2026年3月

---

## 1. 设计概述

### 1.1 设计目标
为宠物主人打造一款温馨、友好、易用的宠物健康管理+社区小程序，帮助用户便捷管理宠物健康档案、获取医疗资源、参与社区互动。

### 1.2 目标用户
- 宠物主人（养猫、养狗为主）
- 养宠新手及资深铲屎官
- 年龄分布：20-45岁，以女性用户为主

### 1.3 设计原则
- **温馨友好**：暖色调为主，图标圆润亲和，营造家的感觉
- **简洁清晰**：信息层级分明，操作路径短，减少认知负担
- **信任可靠**：专业感的医疗信息展示，建立用户信任
- **情感连接**：突出宠物与主人的情感元素，增加用户粘性

### 1.4 色彩体系

| 用途 | 颜色名称 | 色值 | 适用场景 |
|------|----------|------|----------|
| 主色 | 暖心橙 | `#FF8C42` | 主要按钮、图标、Tab选中态 |
| 辅助色 | 清新绿 | `#4CAF50` | 健康相关、成功状态、徽章 |
| 强调色 | 活力红 | `#FF6B6B` | 重要提醒、待办事项、警示 |
| 背景色 | 奶白 | `#FFF9F5` | 页面主背景 |
| 卡片背景 | 纯白 | `#FFFFFF` | 卡片、弹窗 |
| 文字主色 | 深棕 | `#3D2914` | 标题、重要文字 |
| 文字次色 | 灰色 | `#8B7355` | 副标题、描述文字 |
| 文字辅助 | 浅灰 | `#B8A99A` | 提示文字、占位符 |
| 分割线 | 淡粉 | `#F5EDE8` | 列表分割、边框 |

### 1.5 字体规范

| 用途 | 字号 | 字重 | 行高 |
|------|------|------|------|
| 页面标题 | 20px | Bold (700) | 28px |
| 卡片标题 | 16px | SemiBold (600) | 24px |
| 正文 | 14px | Regular (400) | 22px |
| 辅助说明 | 12px | Regular (400) | 18px |
| 按钮文字 | 16px | SemiBold (600) | 24px |

### 1.6 圆角与间距

- **圆角**：卡片 16px，按钮 24px，小元素 8px，头像/图片 12px
- **间距**：页面边距 16px，卡片内边距 16px，列表项间距 12px，元素间距 8px
- **阴影**：卡片 `0 2px 12px rgba(61, 41, 20, 0.08)`

- 高度：44px

- 高度：56px（不含安全区）

- 线性图标为主，线条粗细 1.5px

## 3. 页面设计

### 3.1 用户系统

#### 3.1.1 登录页
- **顶部**：品牌Logo + 温馨Slogan（"爱宠健康，从这里开始"）
- **背景**：奶白色，带有浅色宠物剪影装饰
- **主体**：
  - 手机号输入框 + 验证码输入框 + 获取验证码按钮
  - 微信快捷登录按钮（一键登录）
  - "还没有账号？去注册" 链接
- **底部**：用户协议与隐私政策提示

#### 3.1.2 注册页
- **顶部**：返回箭头 + "注册" 标题
- **主体**：
  - 手机号 + 验证码（复用登录样式）
  - 设置密码 + 确认密码
  - 昵称输入框 + 头像上传区域（圆形，点击上传）
  - 养宠经验选择（单选：新手/有经验/资深）
- **底部**：注册按钮 + 已有账号登录链接

#### 3.1.3 个人信息页
- **头部**：
  - 背景渐变（浅橙到浅粉）
  - 头像（圆形，80px，带相机编辑图标）
  - 昵称 + 养宠经验徽章
- **内容区**：
  - 我的宠物（卡片入口，显示宠物头像列表）
  - 我的收藏（图标 + 文字）
  - 我的问答（图标 + 文字）
  - 设置（图标 + 文字，包括通知设置、隐私设置、关于）
- **底部**：退出登录按钮

---

### 3.2 宠物档案

#### 3.2.1 宠物列表页
- **顶部**：
  - "我的毛孩子" 标题 + 添加宠物按钮（+图标）
- **宠物卡片列表**：
  - 宠物头像（大图，圆角）
  - 名字 + 品种标签
  - 年龄、体重信息
  - 健康状态徽章（健康/需关注/提醒）
- **空状态**：温馨插画 + "还没有添加宠物，去添加吧" + 添加按钮

#### 3.2.2 宠物详情页
- **头部**：
  - 大幅宠物照片（可轮播）
  - 宠物名字 + 品种标签
  - 编辑资料按钮
- **信息卡片**：
  - 基本信息：性别、年龄、生日、体重、芯片号
  - 健康档案：疫苗、驱虫、体检、就医记录入口（带小红点提醒）
- **健康日历**：当月健康提醒日历
- **底部**：健康记录快速添加按钮

#### 3.2.3 添加/编辑宠物页
- **顶部**：返回 + "添加宠物"/"编辑资料"
- **表单区域**：
  - 宠物头像上传（支持多张）
  - 名字输入框
  - 品种选择（联动：猫/狗 → 具体品种）
  - 性别选择（公/母/已绝育）
  - 出生日期选择器
  - 体重输入框
  - 芯片号输入框（选填）
- **底部**：保存按钮

#### 3.2.4 健康记录页（通用结构）
- **顶部**：返回 + "疫苗记录"/"驱虫记录"/"体检记录"/"就医记录"
- **记录列表**：
  - 日期
  - 项目名称
  - 医院/医生（若有）
  - 备注
  - 提醒开关（设置下次提醒）
- **底部**：添加记录按钮（固定底部）
- **空状态**：无记录时显示说明 + 添加入口

---

### 3.3 社区

#### 3.3.1 医院列表页
- **顶部搜索栏**：搜索框 + 筛选按钮（地区、评分）
- **分类Tab**：全部 / 猫专科 / 狗专科 / 24小时 / 定点医保
- **医院卡片**：
  - 医院Logo/门头图
  - 医院名称 + 评分（星星 + 数字）
  - 地址 + 距离
  - 特色标签（24小时、医保定点等）
  - 评价数量
- **底部**：地图模式入口（悬浮按钮）

#### 3.3.2 医院详情页
- **头部**：医院图片轮播 + 收藏按钮
- **信息区**：
  - 医院名称 + 评分
  - 地址 + 电话 + 营业时间
  - 特色服务标签
- **医生团队**（横向滚动）：
  - 医生头像 + 名字 + 职称 + 擅长
- **评价区**：
  - 评分概览（总评分 + 各维度评分柱状图）
  - 评价列表（用户头像 + 昵称 + 评分 + 内容 + 时间 + 点赞）
  - 全部评价入口
- **底部**：我要评价 + 拨打电话 + 导航到店

#### 3.3.3 发布评价页
- **顶部**：返回 + "评价医院"
- **评分区**：
  - 总体评分（星星选择 1-5）
  - 维度评分（环境、态度、技术、服务）
- **内容区**：
  - 评价文字输入框（支持表情）
  - 图片上传（最多9张）
- **标签选择**：推荐项目标签
- **底部**：发布按钮

#### 3.3.4 问答列表页
- **顶部搜索**：搜索问题
- **分类Tab**：全部 / 健康 / 喂养 / 行为 / 其他
- **问答卡片**：
  - 问题标题
  - 摘要（最多2行）
  - 标签（宠物类型 + 问题类型）
  - 回答数 + 收藏数
  - 提问者信息（头像 + 昵称 + 时间）
- **底部**：提问按钮（固定）

#### 3.3.5 问答详情页
- **问题区**：
  - 问题标题（大字）
  - 问题详情
  - 标签 + 提问时间
  - 提问者信息
- **回答列表**：
  - 回答者信息（头像 + 昵称 + 身份标识如"医生"）
  - 回答内容
  - 点赞 + 收藏 + 评论数
  - 展开/收起（长回答）
- **底部**：我来回答（输入框形态）

#### 3.3.6 故事分享页
- **入口**："毛孩子日记"Tab
- **发布器**：发布按钮 → 图片/视频选择 → 文字编辑 → 发布
- **信息流**：
  - 用户头像 + 昵称 + 发布时间
  - 内容（图片/视频 + 文字）
  - 互动栏（点赞 + 评论 + 收藏 + 分享）
  - 评论预览（展开更多评论）
- **详情页**：点击进入全屏查看，支持大图/视频播放

## 4. 交互规范

### 4.1 手势操作
- **下拉刷新**：所有列表页支持
- **上拉加载**：列表分页
- **左滑删除**：健康记录删除
- **长按**：图片/评价收藏
- **双击**：点赞

### 4.2 动画效果
- **页面切换**：右滑进入，左滑退出（300ms）
- **弹窗**：从底部升起（250ms，ease-out）
- **按钮点击**：缩放 0.95（100ms）
- **列表项**：依次淡入（stagger 50ms）
- **Tab切换**：切换指示条平滑移动

### 4.3 加载状态
- **骨架屏**：列表页、详情页
- **Loading图标**：操作中的按钮loading
- **空状态**：统一插画 + 提示文案 + 操作入口

### 4.4 反馈
- **成功**：绿色对勾 + 提示文字（2s后消失）
- **失败**：红色感叹号 + 错误描述
- **警告**：橙色提示 + 引导操作
