## 1. 产品概述

### 1.1 产品定位

宠物健康管理+社区是一款面向宠物主人的综合微信小程序，旨在为用户提供宠物档案全生命周期管理、附近宠物医院查找与评价、健康知识问答以及宠物故事分享的一站式服务。

### 1.2 目标用户

| 用户类型 | 特征 | 核心需求 |
|---------|------|---------|
| 新手宠主 | 初次养宠，缺乏经验 | 宠物喂养指导、健康提醒、附近医院查询 |
| 资深宠主 | 养宠多年，有经验 | 宠物档案管理、健康记录、社交分享 |
| 潜在宠主 | 考虑养宠 | 养宠知识了解、医院评价参考 |

### 1.3 产品愿景

成为宠物主人日常养宠的必备工具，打造「养宠就用XX」的品牌认知。

### 1.4 关键指标 (KPIs)

- 注册用户数：MVP阶段目标 5,000+
- 宠物档案创建率：>60%
- 日活用户：>500
- 社区UGC内容：>200条

## 3. 功能需求详述

### 3.1 用户系统

#### 3.1.1 注册/登录

**优先级：P0（MVP必做）**

| 功能点 | 描述 | 依赖 |
|--------|------|------|
| 手机号+验证码登录 | 输入手机号，获取验证码登录 | 短信服务 |
| 微信一键登录 | 微信授权获取手机号（button open-type="getPhoneNumber"） | 微信开放平台 |
| 游客模式 | 允许游客浏览，限制发帖和收藏 | 无 |
| 登录态维护 | Token机制，7天有效期内自动登录 | Redis |

**流程：**
1. 用户进入小程序 → 检测本地Token
2. 有Token → 自动登录 → 进入首页
3. 无Token → 展示登录页
4. 用户选择登录方式 → 验证 → 登录成功 → 进入首页

#### 3.1.2 用户信息完善

**优先级：P0（MVP必做）**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 昵称 | String | 是 | 2-20字符 |
| 头像 | Image | 否 | 微信头像/上传 |
| 性别 | Enum | 否 | 男/女/保密 |
| 地区 | String | 否 | 省市区选择 |
| 养宠经验 | Enum | 否 | 新手/1-3年/3-5年/5年+ |

**触发场景：** 首次登录后引导完善资料，可跳过

#### 3.2.4 健康提醒

**优先级：P0（MVP必做）**

| 提醒类型 | 触发逻辑 | 提前提醒 |

| 疫苗提醒 | 下次接种日期前7天 | 推送通知 |
| 驱虫提醒 | 下次驱虫日期前7天 | 推送通知 |

### 3.3 社区

#### 3.3.1 医院评价社区

**优先级：P1（重要）**

| 功能 | 描述 |
|------|------|
| 发布评价 | 文字(20-500字)+图片(1-9张)+评分(1-5星) |
| 评价标签 | 服务态度、环境卫生、医生专业度、价格合理 |
| 医院关联 | 评价必须关联一家医院 |
| 展示位置 | 医院详情页评价区 |
| 筛选排序 | 按时间/评分/点赞数 |

#### 3.3.2 医院搜索/筛选

**优先级：P0（MVP必做）**

| 功能 | 描述 |
|------|------|
| 关键词搜索 | 医院名称、地址 |
| 定位筛选 | 距离范围：1km/3km/5km/10km |
| 评分筛选 | 1-5星 |
| 科室筛选 | 内科/外科/眼科/牙科/24小时等 |
| 排序方式 | 距离/评分/销量(评价数) |

#### 3.3.3 医院详情页

**优先级：P0（MVP必做）**

| 模块 | 内容 |
|------|------|
| 基本信息 | 名称、地址、电话、营业时间、评分 |
| 地图展示 | 腾讯地图定位+导航 |
| 科室 | 医院开设的科室列表 |
| 医生 | 主治医生简介（可选展示） |
| 评价 | 用户评价列表（见3.3.1） |
| 收藏 | 收藏医院 |
| 分享 | 分享给好友 |

#### 3.3.4 健康问答社区

**优先级：P1（重要）**

| 功能 | 描述 |
|------|------|
| 发布问题 | 文字(20-500字)+图片(可选) |
| 分类 | 健康/喂养/行为/其他 |
| 回答 | 用户回答，采纳最佳答案 |

| 互动 | 点赞、评论、收藏、分享 |
| 展示 | 社区首页信息流 |

#### 3.3.6 社区互动

**优先级：P1（重要）**

| 互动类型 | 说明 |
|---------|------|
| 点赞 | 评价/问答/故事均可点赞 |
| 评论 | 二级评论（回复评论） |
| 收藏 | 医院/评价/问答/故事 |

### 3.4 AI功能

#### 3.4.1 健康提醒引擎

**优先级：P0（MVP必做）**

| 功能 | 描述 |
|------|------|
| 智能提醒 | 根据宠物档案自动生成疫苗/驱虫/体检提醒 |
| 个性化建议 | 基于宠物品种、年龄给出养护建议 |
| 异常预警 | 就医记录异常时提醒就医 |

**技术实现：**
- 定时任务扫描宠物档案
- 计算下次提醒日期
- 通过微信服务通知推送

#### 3.4.2 基础问诊助手

**优先级：P1（重要）**

| 功能 | 描述 |
|------|------|
| 智能问答 | 基于宠物症状给出初步建议 |
| 常见问题 | 预设100+常见问题FAQ |
| 紧急提示 | 严重症状提示立即就医 |
| 医院推荐 | 根据症状推荐相关医院 |

**边界：**
- 不提供正式医疗诊断
- 明确提示"本回答仅供参考，不替代兽医诊断"
